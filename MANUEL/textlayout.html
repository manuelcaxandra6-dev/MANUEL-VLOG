CSS Layout: Box Model, Float, and Offsets

Learn the fundamental CSS concepts for positioning and spacing elements on a webpage.

1. Box Model

The CSS box model describes the structure of every HTML element as a rectangular box consisting of:

  • Content: The text, image, or media inside the box.
  • Padding: Space between the content and the border.
  • Border: The edge surrounding the padding.
  • Margin: Space outside the border separating it from other elements.

Example:

div { padding: 10px; border: 2px solid black; margin: 20px; }

2. Float

The float property is used to position elements to the left or right of their container, allowing other content to wrap around them.

  • float: left; – Moves the element to the left.
  • float: right; – Moves the element to the right.
  • clear: Used to prevent elements from wrapping around a floated element.

Example:

img { float: left; margin-right: 10px; }

3. Offsets (Positioning)

Offsets are controlled with the position property, which determines how an element is positioned on the page:

  • static: Default position; elements follow normal flow.
  • relative: Positioned relative to its normal position. Use top, right, bottom, left to adjust.
  • absolute: Positioned relative to the nearest positioned ancestor.
  • fixed: Stays in the same place even when scrolling.
  • sticky: Switches between relative and fixed based on scroll position.

Example:

div { position: relative; top: 20px; left: 10px; }
