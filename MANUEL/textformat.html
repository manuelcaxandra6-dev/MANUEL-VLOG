HTML Text Formatting

Learn how to style and structure text using HTML formatting tags.

Common Text Formatting Tags

HTML provides several tags to format text for emphasis, style, or structure. Here’s a summary:

  • – Bold text. Use: Highlight important words.
  • – Important text, usually bold. Use: Emphasize meaning, accessible for screen readers.
  • – Italic text. Use: Highlight terms, foreign words, or technical terms.
  • – Emphasized text, usually italic. Use: Convey stress or importance in a sentence.
  • – Highlighted text. Use: Draw attention to specific parts of content.
  • – Smaller text. Use: For fine print or secondary information.
  • – Strikethrough text. Use: Indicate removed or outdated content.
  • – Inserted text, usually underlined. Use: Show new or added content.
  • – Subscript text. Use: Chemical formulas, math indices.
  • – Superscript text. Use: Exponents, footnotes, ordinal numbers.
