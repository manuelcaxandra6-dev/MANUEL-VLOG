Introduction to CSS

Learn how to style your webpages and make them visually appealing using CSS.

CSS, or Cascading Style Sheets, is a language that defines how HTML elements should appear on a webpage. It separates content from design, allowing you to control colors, fonts, layouts, spacing, and overall presentation.

  • CSS can be applied inline, inside a <style> tag, or via an external stylesheet linked to your HTML.
  • Selectors target HTML elements to apply specific styles, giving you control over how each element looks.
  • Properties define what aspect of an element to style, and values determine the exact styling.
  • Using CSS makes webpages more organized, attractive, and responsive across different devices.

CSS Syntax Example

Part Description
Selector The element(s) you want to style, such as h1, p, .class, or #id.
Property The style feature you want to change, like color, font-size, or margin.
Value The specific setting for the property, such as red, 16px, or bold.
Example h1 { color: blue; font-size: 24px; }
