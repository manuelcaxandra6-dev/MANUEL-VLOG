CSS Basics: Selectors & Attributes

Learn how to target elements and apply styles using CSS.

CSS Selectors & Elements

Selectors allow you to choose which HTML elements to style. Here are some common ones:

  • Div – A container element used to group other elements. Use: for layout sections or styling a block of content.
  • Span – An inline container for text or small elements. Use: to style part of a sentence or inline content.
  • Class – A reusable identifier for one or more elements. Use: apply the same style to multiple elements using class="name".
  • ID – A unique identifier for a single element. Use: target a specific element with id="name".
  • Links – Anchor tags that navigate to other pages or sections. Use: style link colors, hover effects, and visited states.

Basic CSS Attributes

These properties control how elements appear on your webpage:

Attribute Description Example
color Sets the text color of an element. color: red;
background-color Sets the background color of an element. background-color: #f0f0f0;
font-size Changes the size of the text. font-size: 16px;
font-family Specifies the font type for text. font-family: Arial, sans-serif;
margin Sets space outside an element. margin: 10px;
padding Sets space inside an element, between content and border. padding: 8px;
border Adds a border around an element. border: 1px solid black;
text-align Aligns text inside an element. text-align: center;
display Specifies how an element is displayed (block, inline, inline-block, etc.). display: block;
